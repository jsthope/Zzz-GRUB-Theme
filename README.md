## Installation

```bash
cd Zzz-grub-theme
sudo ./install.sh
```

## Contents

- Theme installed to `/boot/grub*/themes/assets`
- Background image: `assets/background.png`
- Font generated during installation from `assets/original.ttf`
- GRUB timeout updated in `/etc/default/grub`

## Uninstallation

```bash
cd Zzz-grub-theme
sudo ./uninstall.sh
```
